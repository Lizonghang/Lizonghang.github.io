儿童锁屏幕监测报警

发表于 2016-07-15   |   分类于 Python

这是一个丧尽天良的程序，用于家长对孩子使用电脑时的监控，当程序发现屏幕上出现了疑似色情的图片/视频时，将屏幕进行截图保存并邮件通知家长，也能判断出程序是否正常退出。可以设置采样间隔与监控时间，可以设置截图保存路径与图片尺寸。

假设文件名为ScreenScrape.py

运行程序：

  • path - 存放截图的文件夹，必选
  • Width - 存储截图时的截图宽度，可选
  • Height - 存储截图时的截图高度，可选
  • duration - 采样周期，单位min，可选
  • totalTime - 监控时长，单位min，可选
多张图片合成GIF图
Mac上安装python-opencv
