算法导论14章3节答案

发表于 2017-07-14   |   分类于 算法导论

14.3-1 写出作用于区间树的结点且在O(1)时间内更新max属性的过程LEFT-ROTATE的伪代码。

14.3-2 改写INTERVAL-SEARCH的代码，使得当所有区间都是开区间时，它也能正确地工作。

14.3-3 请给出一个有效的算法，对一个给定区间i，返回一个与i重叠且具有最小低端点的区间；或者当这样的区间不存在时返回T.nil。

14.3-4 给定一棵区间树T和一个区间i，请描述如何在O(min(n, klgn))时间内列出T中所有与i重叠的区间，其中k为输出的区间数。（提示：一种简单的方法是做若干次查询，并且在这些查询的操作中修改树，另一种复杂点的方法是不对树进行修改）

14.3-5 对区间树和一个区间i，请修改有关区间树的过程来支持新的操作INTERVAL-SEARCH-EXACTLY(T,i)，它返回一个指向T中节点x的指针，使得x.int.low=i.low且x.int.high=i.high；或者，如果T不包含这样的区间时返回T.nil。所有的操作（包括INTERVAL-SEARCH-EXACTLY）对于包含n个结点的区间树的运行时间都应该为O(lgn)。

14.3-6 说明如何来维护一个支持操作MIN-GAP的一些数的动态集Q，使得该操作能给出Q中两个最接近的数之间的差值。例如，Q={1,5,9,15,18,22},则MIN-GAP返回18-15=3，因为15和18是Q中两个最接近的数。要使得操作INSERT、DELETE、SEARCH和MIN-GAP尽可能高效，并分析它们的运行时间。

思路：在每个结点维护附加信息min_gap差值域、successor指针域、predecessor指针域。

  • x.min_gap记录了以x为根节点的树的min_gap。
  • x.successor记录了x在树中的后继结点。
  • x.predecessor记录了x在树中的前驱结点。

14.3-7 VLSI数据库通常将一块集成电路表示成一组矩形，假设每个矩形的边都平行于x轴或y轴，这样可以用矩形的最小和最大的x轴与y轴坐标来表示一个矩形。请给出一个O(nlgn)时间的算法，来确定n个这种表示的矩形集合中是否存在两个重叠的矩形。算法不一定要输出所有重叠矩形，但对于一个矩形完全覆盖另一个（即时边界不相交），一定能给出正确的判断。（提示：移动一条“扫描线”，穿过所有的矩形。）

算法过程：

  1. 将矩形投影到x轴得到矩形在x轴上的二维区间数组并展开为一维点数组。
  2. 将一维点数组进行升序排序。
  3. 逐个取出x点，判断其为某个区间的低端点或高端点：
    • 若为低端点，获取该x区间对应y区间，在区间树中进行搜索查找是否有重叠的区间，若存在，则可退出或记录到列表中。随后将该y区间插入区间树中。
    • 若为高端点，获取该x区间对应y区间，并将该y区间从区间树中删除。
  4. 若退出循环，则返回T.nil或记录列表。
